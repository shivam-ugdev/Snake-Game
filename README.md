# Snake Game
A simple Snake game developed using Unity and C#.
The player controls a snake that grows in size by eating food. The game includes a scoring system, high score tracking, a start panel, pause functionality, and a game over screen. This project was created to practice basic game mechanics, UI handling, and game state management.
## Features
- Classic Snake gameplay
- Snake size increases when food is collected
- Score and high score system
- Start panel
- Pause button and pause panel
- Game over panel
